
# From crates.io

Rust must be installed: [https://rustup.rs](https://rustup.rs)

Run

    cargo install ddoc --locked

# From sources

Rust must be installed: [https://rustup.rs](https://rustup.rs)

Clone the repository:

    git clone https://github.com/Canop/ddoc.git

Move to its directory:

    cd ddoc

Compile and install:

    cargo install --path .
